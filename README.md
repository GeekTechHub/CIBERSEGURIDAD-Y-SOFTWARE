---

## 🏗️ Arquitectura del Repositorio
He estructurado este espacio utilizando técnicas de **Git Subtree** para mantener la integridad del historial de cada módulo independiente, centralizándolos para una gestión eficiente.

### 🔐 Ciberseguridad & Forense
* **`herramientas-de-penetracion/`**: Scripts y utilidades para auditorías de red (Airodump-ng, etc.).
* **`herramientas-de-informatica-forense/`**: Metodologías y software para preservación de evidencia.
* **`VPN/`**: Configuraciones y protocolos de conectividad segura.
* **`comandos-del-cmd/`**: Diccionario técnico de administración de sistemas vía CLI.

### 🌐 Redes & Conectividad
* **`redes/`**: Fundamentos, topologías y prácticas de networking.
* **`conectividad-jnc/`**: Scripts automatizados de prueba de conectividad y diagnóstico.

### 💻 Ingeniería de Software & Desarrollo
* **`node-api/`**: Implementación de APIs RESTful utilizando arquitectura asíncrona en Node.js.
* **`programacion-orientada-a-objetos/`**: Paradigmas de diseño de software escalable.
* **`estructura-datos/` & `algoritmos/`**: Implementaciones de lógica computacional optimizada.
* **`base-datos/`**: Modelado, consultas y gestión de sistemas de bases de datos.
* **`linux/`**: Administración de sistemas, personalización de kernel y Bash scripting.
